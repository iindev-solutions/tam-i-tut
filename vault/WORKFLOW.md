# Vault Workflow

## Session Start

Read these files in order:

1. `vault/master_index.md`
2. `vault/WORKFLOW.md`
3. `vault/sprint.md`
4. `vault/resume-plan.md`

Then read task-specific docs under:

- `vault/CODE_MAP.md`
- `vault/wiki/architecture/`
- `vault/wiki/services/`

## Session End

Before closing meaningful work, update `vault/`.

Mandatory:

1. `vault/logs/changelog.md`

Update when relevant:

1. `vault/resume-plan.md`
2. `vault/sprint.md`
3. `vault/CODE_MAP.md`
4. `vault/SESSION_LEDGER.md`

## Rule

Do not leave important state only in chat.

Move it into `vault/`.
